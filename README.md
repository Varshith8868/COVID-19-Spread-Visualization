# COVID-19 Global Dashboard

A fully interactive, real-time dashboard visualizing global COVID-19 statistics using D3.js.  
This project features an interactive world map, bar chart, pie chart, and a ranked table of countries, all styled with a modern dark UI.
---
## üåç Features

- **Real-time Data:** Fetches live COVID-19 data from the [disease.sh API](https://disease.sh/).
- **Interactive World Map:** Visualizes cases per country using a color-coded choropleth map, with hover tooltips.
- **Top 10 Countries Table:** Displays the most affected countries with sortable statistics.
- **Bar Chart:** Ranks top 10 affected countries with tooltips on hover.
- **Pie Chart:** Shows global distribution of recovered, active, and deceased cases, with interactive tooltips and clear labels.
- **Responsive Design:** Works on desktop and mobile screens, with fluid layout and touch support.
- **Modern UI:** Uses glassmorphism styling with smooth transitions and hover effects.
